## Phase 4: Authentication — COMPLETE

- Task: Auth middleware with JWT, API Key, and OIDC strategies behind common Authenticator interface
- Key decisions:
  - New external dep: github.com/golang-jwt/jwt/v5@v5.3.1
  - Authenticator interface with factory dispatch on provider.Type (jwt/apikey/oidc)
  - Auth middleware reads MatchedRoute from context → looks up provider name → calls Authenticate
  - Provider "none" or nil auth config → skip auth entirely
  - JWT: HMAC/RSA/ECDSA support; key material loaded once at construction; Bearer token from Authorization header
  - API Key: constant-time compare via crypto/subtle.ConstantTimeCompare; configurable header (default X-API-Key)
  - OIDC: JWKS fetching with background refresh (80% of TTL); JWK→RSA/ECDSA parsing; introspection mode via RFC 7662
  - OIDC discovery: fetches .well-known/openid-configuration for jwks_uri if only issuer_url set
  - AuthResult stored in context for downstream middleware (rate limit claim keying)
  - Error responses: JSON {"error":"unauthorized","message":"..."} format
  - JWT error classification: expired/signature/generic mapped to specific messages
- Files created:

## Phase 3: CORS — COMPLETE

- Task: CORS preflight handling, response header injection, per-route override
- Key decisions:
  - No external deps; stdlib only
  - corsResponseWriter intercepts WriteHeader to inject CORS headers at write time (after proxy copies upstream headers)
  - Per-route override uses isOverride flag + internal marker header (X-Gateway-Cors-Override) to prevent global CORS from overwriting; marker deleted before response sent
  - Complete replacement semantics: per-route CORS fully replaces global, no field-level merge
  - Wildcard + credentials edge case: logs warning at construction, uses requesting origin instead of *
  - Preflight = OPTIONS + Origin + Access-Control-Request-Method (all three required); plain OPTIONS forwarded to handler
  - CORS positioned last in global chain (after Metrics, before Router)
  - Preflights use global config only (router hasn't matched yet); per-route overrides apply to regular requests via per-route handler chain
  - Added ExposedHeaders field to CORSConfig, CORS pointer to RouteConfig
- Files created:
  - internal/middleware/cors.go (CORS, CORSOverride, corsResponseWriter, helpers)

## Phase 6: Resilience (Circuit Breaker + Retry) — COMPLETE

- Task: Circuit breaker state machine + retry with exponential backoff + jitter
- Key decisions:
  - No external deps; stdlib only
  - CB state machine: Closed→Open (failure threshold), Open→HalfOpen (timeout elapsed), HalfOpen→Closed (success threshold), HalfOpen→Open (any failure)
  - fn() executes outside the lock to avoid holding mutex during slow upstream calls
  - Execute method: lock→check state→unlock→run fn→lock→record result
  - WithClock option for deterministic testing of timeout transitions
  - OnStateChange callback updates Prometheus gauge (wired in main.go)
  - Retry: body buffered in memory, replayed via bytes.NewReader per attempt
  - Backoff: initialDelay * multiplier^(attempt-1) * jitter where jitter ∈ [0.5, 1.5)
  - Context cancellation checked before sleep and before each attempt
  - httptest.NewRecorder captures response for retry evaluation
  - CB+Retry combo: retry loop wraps each attempt in cb.Execute; ErrCircuitOpen breaks immediately → 503
  - WithCircuitBreaker: CB-only wrapper (no retry), treats status >= 500 as failure
  - Retry without CB: cb=nil handled gracefully
  - Moved metrics/registry creation before handler loop in main.go so CB callback can reference metrics
- Files created:
  - internal/middleware/circuitbreaker/circuitbreaker.go (State, Execute, options)
  - internal/middleware/circuitbreaker/circuitbreaker_test.go (11 tests incl. concurrency)
  - internal/middleware/retry/retry.go (Retry, WithCircuitBreaker, sleep/jitter options)
  - internal/middleware/retry/retry_test.go (17 tests incl. 2 integration)
- Files modified:
  - cmd/gateway/main.go (import CB+retry, wire per-route, moved metrics init earlier)
- All validations pass: gofmt, go test ./... -race, go vet ./...
